# 📊 Attendance Analyzer

A smart rule-based system to analyze and predict student attendance.

## 🚀 Features

- Calculates attendance percentage
- Shows status (Safe / Warning / Danger)
- Predicts future attendance
- Suggests classes needed to reach 75%
- Visual graph for attendance trend

## 🛠️ Tech Used

- Python
- Streamlit
- Matplotlib

## ▶️ How to Run

```bash
pip install streamlit matplotlib
streamlit run app.py
